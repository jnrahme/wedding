Joey & Ana Wedding Invitation
=============================

A mobile-first Next.js wedding invitation for Joey Rahme and Ana Maria Lahoud.

Local Development
-----------------

```bash
cd wedding-invitation
npm install
npm run generate:assets
npm run dev
```

Open `http://localhost:3000`.

Production Build
----------------

```bash
npm run build
npm run start
```

Editing Wedding Details
-----------------------

All primary content is in `config/wedding.ts`, including:

- Music file URL
- Couple and venue photos
- Wedding schedule
- Google Maps links
- Travel information
- Contact information
- SEO and social-sharing metadata

Assets
------

Placeholder images and the original placeholder instrumental track are generated by:

```bash
npm run generate:assets
```

Replace files in `public/images` and `public/audio` with final licensed or original assets. Keep file paths the same, or update `config/wedding.ts`.

Deploying to Vercel
-------------------

1. Push this repository to GitHub.
2. In Vercel, choose **Add New Project**.
3. Select the repository.
4. Set the root directory to `wedding-invitation`.
5. Use the default Next.js settings.
6. Deploy.

Custom Domain on Vercel
-----------------------

1. Open the Vercel project.
2. Go to **Settings → Domains**.
3. Add the custom domain.
4. Follow Vercel's DNS instructions at your domain registrar.
5. After DNS verifies, update `siteUrl` in `config/wedding.ts`.
